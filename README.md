


# Calculator App

This is a simple calculator application that takes in 2 integers as input and give us their addition, subtraction, multiplication and the remainder.


# Concepts used

- UITextField
- UIButton
- UILabel
- UITapGestureRecognizer
- Life Cycle Methods
- Optionals, Optional Binding
